🎨 Avatar Generator – Генератор аватарок на 7 языках
Создавайте уникальные аватарки на основе текста или случайных параметров.
Поддерживает настройку цвета, размера, формы, добавление текста, рамок и фонов.
Реализован на 7 языках программирования с единым интерфейсом командной строки.

🚀 Возможности
Генерация на основе текста – из имени или слова создаётся уникальный идентификатор, определяющий цвета и форму.

Настройка параметров:

Размер изображения (по умолчанию 256×256).

Формат (PNG, JPG, WEBP).

Цветовая схема (автоматическая или заданная).

Форма (круг, квадрат, шестиугольник).

Добавление текста (инициалы или полное имя).

Рамка и фон.

Пакетная генерация – создание нескольких аватарок с разными именами.

Случайная генерация – создание уникальной аватарки без параметров.

Цветной вывод – информативные сообщения в терминале.

Кроссплатформенность – работает на Windows, Linux, macOS.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [имя] [опции]
Аргументы
Аргумент	Описание
[имя]	Имя или текст для генерации аватарки (по умолчанию случайное)
Опции
Опция	Описание
-o, --output <файл>	Выходной файл (по умолчанию avatar.png)
-s, --size <N>	Размер в пикселях (по умолчанию 256)
-f, --format <png|jpg|webp>	Формат (по умолчанию png)
-c, --color <цвет>	Основной цвет (HEX или имя, например #FF5733)
-b, --bg <цвет>	Цвет фона (по умолчанию случайный)
-t, --text	Добавить текст (имя или инициалы)
-r, --random	Случайная генерация (игнорирует имя)
-h, --help	Справка
Примеры
bash
# Сгенерировать аватарку на основе имени "Алексей"
python avatar.py Алексей

# Создать круглую аватарку с заданным цветом
python avatar.py Мария -c #3498db -s 512 -t

# Пакетная генерация для списка имён
python avatar.py batch -n "Анна,Борис,Светлана"

# Случайная аватарка
python avatar.py -r -o random_avatar.png

🧠 Алгоритм генерации
Хеширование имени – строка преобразуется в числовой хеш, который используется для генерации цветов и формы.

Цвета – основной и фоновый цвета вычисляются на основе хеша (если не заданы явно).

Форма – рисуется круг, квадрат или шестиугольник с заливкой основным цветом.

Текст – если включено, на аватарку наносятся инициалы или полное имя.

Рамка – добавляется контур (опционально).

Сохранение – изображение сохраняется в указанном формате.

✨ Дополнительные фичи
Пакетная генерация – создание аватарок для списка имён из файла или через запятую.

Автоопределение формата – по расширению выходного файла.

Цветовая палитра – предустановленные схемы для разных типов имён.
